Building a Dockerized Web Application with MySQL Database

Tools Used:

  • HTML
  • CSS
  • JavaScript
  • Node.js
  • MySQL
  • Docker
  • Apache HTTP Server

Project Steps:

  1. Building the Frontend:

    • Create HTML, CSS, and JavaScript files for the frontend.
    • Set up pages: Homepage, About us (receives data from the database and displays in a slider), and About the project.

  2. Creating the Frontend Dockerfile:

    • Copy frontend files into Apache server root directory.
    • Expose port 80.
    • Run Apache server in the foreground.

  3. Creating the Database Scripts:

    • Create MySQL script file (schema.sql) containing database schema and initial data.

  4. Creating the Database Dockerfile:

    • Copy database script file(s) into /docker-entrypoint-initdb.d/ folder within MySQL container.

  5. Creating the Backend:

    • Set up a Node.js Express server.
    • Connect Express server to MySQL database.
    • wait for the database port to become available, and then start the server to listen for incoming HTTP requests.

  6. Creating the Backend Dockerfile:

    • Install Node.js dependencies using npm.
    • Copy backend files into Docker container.
    • Expose port 3000.
    • Start the Node.js application.

  7. Creating Docker Compose Configuration:

    • Define services for frontend, backend, and MySQL database.
    • Connect all services to the same network.
    • Specify dependencies between services.
    • Configure volumes and ports as needed.

  8. Building and Running the Docker Compose Environment:

    • Run docker compose up --build -d to build and start Docker containers.
    • Access the application at http://localhost:9080 .
